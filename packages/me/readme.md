# uebersicht-me

> Me widget for [Übersicht](http://tracesof.net/uebersicht)

## Features

- Display personal information or status
- Supports light and dark color schemes
- Mona Sans typography for consistency
- SVG icon support with color theming

## Installation

1. Clone this repository into your Übersicht widgets folder:

```bash
git clone https://github.com/stoe/uebersicht.git ~/.config/übersicht/widgets/uebersicht
```

2. Or manually copy the `packages/me` folder into your Übersicht widgets directory.

## Styling

The widget uses the following design tokens:

- **Font**: Mona Sans with sans-serif fallback
- **Light mode**: #101411
- **Dark mode**: #f2f5f3
